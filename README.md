## вңЁ Features

### Core Platform
- рҹ“қ **Posts** - Text, images, videos, polls, articles
- рҹ”— **Federation** - Full ActivityPub support, connect with Mastodon
- рҹ”’ **E2E Encryption** - Private messages encrypted end-to-end
- рҹ‘Ҙ **Followers/Following** - Standard social graph
- рҹ”” **Notifications** - Real-time notifications
- рҹ”Қ **Discovery** - Trending posts, hashtags, profiles

### Privacy
- рҹҡ« **No Ads** - Completely ad-free
- рҹ”җ **Privacy Controls** - Granular post visibility
- рҹ—‘пёҸ **Data Portability** - Export all your data
- рҹ”‘ **Key Management** - User-controlled encryption keys
- рҹҸіпёҸ **No Tracking** - No analytics, no cookies

### Moderation
- рҹӣЎпёҸ **Content Moderation** - AI-powered content flagging
- рҹҡ« **Block/Domain Block** - User and server blocking
- рҹ“Ӣ **Moderation Tools** - For admins and users
- рҹ”Ё **Appeal System** - Moderation decision appeals

## рҹ“Ұ Installation

```bash
git clone https://github.com/moggan1337/DecentraPost.git
cd DecentraPost

# Install dependencies
npm install

# Setup environment
cp .env.example .env
# Configure database, Redis, S3

# Initialize database
npm run db:migrate

# Start development
npm run dev
```
